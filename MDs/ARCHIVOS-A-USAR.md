## 🚀 Cómo usar el proyecto

1. Abre **uni-organizado.html** en tu navegador
2. Todos los módulos JavaScript se cargarán automáticamente desde la carpeta `js/`
3. Los estilos se aplicarán desde **uni-styles.css**
